var, let, const를 사용하여 생성된 변수들의 차이점

실무에서 변수를 선언할 때, 값의 재할당이 필요한 경우 let을, 아닌 경우 const를 사용했다. var를 사용하는 일은 거의 없었다.
var, let, const에 대해서 술술 말할 수 있도록 정리를 해보았다.

1. var 은 함수 스코프, let과 const 는 블록스코프

2. var로 선언한 변수는 호이스팅되고 let, const는 그렇지 않다.

3. var 는 변수의 재선언을 허용하고 let,const는 허용하지 않는다.

4. let으로 선언한 변수는 값을 재할당할 수 있지만 const는 불가능하다.

하지만 const로 생성한 변수가 object의 경우, object의 프로퍼티는 수정가능하다.

만약 object property의 값도 상수로 만들고 싶다면, Object.freeze() 를 이용해 object를 수정불가능하게 한다.
