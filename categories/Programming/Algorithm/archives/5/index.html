탐욕 알고리즘( greedy-algorithm )

  • 불가능한 문제, 즉 빠른 알고리즘 해법이 존재하지 않는 NP-완전 문제를 다루는 법을 배우자
  • 문제 해결이 불가능한지 아닌지 파악하는 법을 배우고, NP-완전 문제에 대한 간략한 해법을 빨리 구할 수 있는 근사 알고리즘도 배우자
  • 아주 간단한 문제 해결 기법의 하나인 탐욕 알고리즘을 공부하자!

다익스트라 알고리즘(dijkstra-algorithm)

  • 그래프의 간선에 가중치를 준 가중 그래프weighted graph을 배우자.
  • 가중 그래프에서 X까지의 최단 경로를 구하는 다익스트라 알고리즘을 배우자.
  • 그래프에 사이클 cycle이 있을 경우 다익스트라 알고리즘을 사용할 수 없다.
