버블 정렬 vs 선택 정렬 vs 삽입 정렬

sorting이란, 데이터가 주어졌을 때 이를 정해진 순서대로 나열하는 것이다.

대표적인 sorting 알고리즘인 버블 정렬과 선택 정렬 그리고 삽입 정렬의 차이점을 알아보고 직접 구현해보자.

버블 정렬

두 인접한 데이터를 비교해서, 앞에 있는 데이터가 뒤에 있는 데이터보다 크면, 자리를 바꾸는 정렬 알고리즘

선택 정렬

  1. 최소값을 선택
  2. 해당 최소값을 데이터 맨 앞에 위치한 값과 교체
  3. 맨 앞의 위치를 뺀 나머지 데이터를 동일한 방법으로 반복

삽입 정렬

특정 자리에 삽입해서 배열을 정렬한다!

  1. 두번째 인덱스(key라고 하자)에 있는 데이터를 선택한다.
  2. key의 바로 앞에 있는 데이터(B)부터 비교해서, key의 값이 더 작으면 B값과 key의 값을 swap
  3. 이 과정을 key의 value < B이 될 때까지 반복.

이 세가지 정렬 모두 for문을 중첩해서 두번 돈다. 시간복잡도는 O(n^2)이 된다.
만약 이미 정렬이 되어 있는 배열이라면 O(n)이다.
