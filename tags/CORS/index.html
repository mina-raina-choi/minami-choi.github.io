크로스 도메인 이슈 해결하기

동일 출처 정책 Same Origin Policy(SOP)이란?

만약 나의 api서버를 외부 웹서비스에서 막 접근해서 가져다 쓴다면 보안상으로도 이슈가 있고 트래픽 과부하 문제도 발생할 수 있다. 따라서 이런 행위를 차단하기 위해 동일한 도메인이 아니면 아래처럼 크로스 도메인 이슈를 발생시킨다.
