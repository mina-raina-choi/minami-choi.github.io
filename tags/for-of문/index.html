for문, foreach문, for~in문, for~of문 차이점

for문, foreach문, forin문, forof문 차이점

for문의 가독성을 좀 더 좋게 하기 위해서 foreach가 나왔지만 아래와 같은 문제가 있었다.

foreach

  1. You can’t break out of this loop using a break statement or move to the next iteration with continue.
    (break로 loop를 빠져나오지 못하고, continue로 그다음 반복으로 이동하지 못함)
  2. You can’t return from the enclosing function using a return statement.
    (return을 사용해 해당 function을 마무리하지 못함)
