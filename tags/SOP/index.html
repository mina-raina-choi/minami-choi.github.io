동일 출처 정책 Same Origin Policy(SOP)이란?

같은 출처에 있는 리소스만 접근할 수 있도록 제한하는 보안정책이다. XMLHttpRequest 나 Fetch api 호출로 데이터를 요청하는 경우에 적용되고 이미지나 css, 동적으로 로드되는 스크립트에는 적용되지 않는다. (단, 웹폰트는 예외)
